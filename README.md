# DocForge-Learning-Tasks

This repository contains the tasks for the learning phase of the project.

---

## 📌 How This Repository Works

* Each task will have its **own folder**.
* Inside every task folder, you will find a **README.md** containing:

  * Task instructions
  * Resources (if any)
  * Expected outcome

---

## ✅ What You Need To Do

1. Complete each task in order.
2. After finishing a task, create a file named **explanation.md** inside that task’s folder.
3. In `explanation.md`, briefly write:

   * Key concepts you learned
   * Any challenges faced (optional)
   * How you approached the problem

This helps us track your progress and ensures you understand the concepts, not just complete the code.

---

## 🤖 Use of AI Tools

You are **allowed to use AI tools** (ChatGPT, Copilot, etc.) for help.

However:

* You must understand the code you submit.
* You should be able to explain your logic clearly in `explanation.md`.
* Do not blindly copy solutions without understanding them.
* If AI helped you significantly, mention how it helped.

The goal is learning, not speed.

If you cannot explain your solution, you have not completed the task.

---

## 💡 Notes

* Push your work regularly to GitHub.
* Keep explanations short, clear, and in your own words.
* Write clean, readable code.
* If you have doubts, ask in the group or DM a mentor :))
